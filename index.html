John Conway's Game of Life

When paused, you can toggle cells on and off by clicking them. Spacebar: pause/unpase
Up and down arrows: Change grid size (up=increase size, down=decrease). Only works when paused
Left and right arrows: Change framerate (right=higher, left=lower)
Backslash [\]: Clear the grid. Only works when paused
Tilde [~] (but without the shift, so actually a backquote `): Randomize the state of the grid. Only works when paused.
Enter: Move forward one generation (one frame). Only works when paused

1-9 and 0 keys (not on the numpad) create different patterns. 1-5 create objects known as oscillators, which cycle between a few shapes. 4-9 create spaceships, which slowly move accross the screen. 0 creates a type of spaceship gun, which steadily creates small spaceships. If the object doesn't show up when you press the key, it means you need to increase the size of the grid.
There are more patterns listed on the wikipedia page for the game of life, as well as on the online game of life's lexicon page
